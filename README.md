# nodecms
基于nodejs的cms系统。
本项目主要用于演示nodejs + express + sequelize + mysql如何开发一个简单的内容管理系统。

项目中，还使用了async来进行同步数据的读取。

**代码比较简陋，主要是为了让大家看到更多的函数和方法使用，真实的项目中不会是这样。**

# 数据库脚本
项目根目录下有一个database.sql文件，此文件是项目的数据库创建文件。
